# 女性主义立场图谱测评（静态站）

## 本地运行
```bash
npm install
npm run dev
```

## 构建
```bash
npm run build
npm run preview
```

## 部署到 GitHub Pages
1. 创建 GitHub 仓库（建议仓库名：feminism-quiz）
2. 把本项目文件全部推到仓库 main 分支
3. 在 GitHub 仓库 Settings -> Pages：
   - Build and deployment 选择 GitHub Actions
4. 等待 Actions 运行完成
5. 访问：`https://<你的用户名>.github.io/<仓库名>/`
